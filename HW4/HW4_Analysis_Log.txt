once I have a sam file I need to sort them and convert to bam files

samtools sort grp1_aligned.sam -o grp1_aligned.sorted.sam
samtools sort grp2_aligned.sam -o grp2_aligned.sorted.sam
samtools sort grp3_aligned.sam -o grp3_aligned.sorted.sam

samtools view -b -o grp1_aligned.sorted.bam grp1_aligned.sorted.sam
samtools view -b -o grp2_aligned.sorted.bam grp2_aligned.sorted.sam
samtools view -b -o grp3_aligned.sorted.bam grp3_aligned.sorted.sam

I can then index the created bam file to create bai files

samtools index grp1_aligned.sorted.bam
samtools index grp2_aligned.sorted.bam
samtools index grp3_aligned.sorted.bam

Now I need to use picard to Add read groups and mark duplicates.

picard AddOrReplaceReadGroups I=grp1_aligned.sorted.bam O=grp1_aligned.sorted.RG.bam RGID=sample RGLB=lib1 RGPL=ILLUMINA RGPU=unit1 RGSM=sample
picard AddOrReplaceReadGroups I=grp2_aligned.sorted.bam O=grp2_aligned.sorted.RG.bam RGID=sample RGLB=lib1 RGPL=ILLUMINA RGPU=unit1 RGSM=sample
picard AddOrReplaceReadGroups I=grp3_aligned.sorted.bam O=grp3_aligned.sorted.RG.bam RGID=sample RGLB=lib1 RGPL=ILLUMINA RGPU=unit1 RGSM=sample


picard MarkDuplicates -I grp1_aligned.sorted.RG.bam -O grp1_aligned.sorted.RG.markdups.bam -M grp1_aligned.sorted.RG.markdups_metrics.txt
picard MarkDuplicates -I grp2_aligned.sorted.RG.bam -O grp2_aligned.sorted.RG.markdups.bam -M grp2_aligned.sorted.RG.markdups_metrics.txt
picard MarkDuplicates -I grp3_aligned.sorted.RG.bam -O grp3_aligned.sorted.RG.markdups.bam -M grp3_aligned.sorted.RG.markdups_metrics.txt 

I can then use qualimap again to check the marked duplicates

qualimap bamqc -nt 4 -bam grp1_aligned.sorted.RG.markdups.bam -outdir grp1_aligned.sorted.RG.markdups_qualimap_results --java-mem-size=4G
qualimap bamqc -nt 4 -bam grp2_aligned.sorted.RG.markdups.bam -outdir grp2_aligned.sorted.RG.markdups_qualimap_results --java-mem-size=4G
qualimap bamqc -nt 4 -bam grp3_aligned.sorted.RG.markdups.bam -outdir grp3_aligned.sorted.RG.markdups_qualimap_results
